# Student Portal Web Application

## Overview

The Student Portal Web Application is a front-end project developed using HTML5 and CSS3. The project focuses on implementing modern layout techniques such as Flexbox and the CSS Box Model to create a structured and visually consistent user interface.

This application demonstrates fundamental web development concepts and serves as a foundational project for beginners.

---

## Features

* Structured and semantic HTML5 layout
* Clean and consistent styling using CSS3
* Implementation of the CSS Box Model
* Flexbox-based header and navigation layout
* Centrally aligned hero section
* Student statistics bar with equal spacing
* Styled course cards with borders and shadows

---

## Layout Description

### Header and Navigation

The header section uses Flexbox to align the site title and navigation links horizontally. The navigation links are evenly spaced and aligned for clarity.

### Hero Section

The hero section is designed using a column-based Flexbox layout. It includes a heading, descriptive text, and a call-to-action button, all centered.

### Student Statistics Bar

A horizontal statistics bar displays key information such as courses enrolled, GPA, and semester. Flexbox is used to distribute space evenly across all items.

### Course Cards

Each course is presented using a card layout styled with padding, borders, and shadows. This demonstrates the effective use of the CSS Box Model.
